# 3D Models

This directory contains unofficial experimental modified 3D parts for my LaLaPad Gen2 haptic/force feedback prototype.

These parts are not official LaLaPad Gen2 parts.  
They are shared as experimental build-log materials only.

## Files

- `modified-parts/pad-lid-1-hapticmod.stl`
- `modified-parts/pad-lid-2-hapticmod_L.stl`
- `modified-parts/pad-lid-2-hapticmod_R.stl`
- `modified-parts/fsr-base-hapticmod_L.stl`
- `modified-parts/fsr-base-hapticmod_R.stl`

## Disclaimer

- These parts are not production tested.
- Print tolerances may affect fit, preload, and FSR behavior.
